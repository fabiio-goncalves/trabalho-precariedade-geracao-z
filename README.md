# Projeto Final – Escrita de Textos Técnicos e Científicos

## 📌 Descrição
Este trabalho foi desenvolvido no âmbito da unidade curricular de Escrita de Textos Técnicos e Científicos.

O projeto aborda a temática da precariedade laboral da Geração Z em Portugal, analisando os seus impactos sociais, económicos e profissionais, bem como possíveis soluções para mitigar este problema.
